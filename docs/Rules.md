# AI Coding Rules

Every AI coding agent working on this project MUST follow these rules.

## Before Writing Code

Always read:

- Design.md
- Brand.md
- Content.md
- Todo.md

---

## Never

- Invent random colors
- Use gradients unless requested
- Change typography
- Create inconsistent spacing
- Use multiple design styles
- Generate generic SaaS layouts
- Add unnecessary dependencies

---

## Always

- Build reusable components
- Use semantic HTML
- Follow accessibility best practices
- Keep animations subtle
- Prioritize performance
- Use TailwindCSS only
- Follow the Design System

---

## Goal

Every new page should look like it belongs to the same premium luxury salon website.

Consistency is more important than creativity.
