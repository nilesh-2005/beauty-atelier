# Beauty Atelier — Design System

## Purpose

This document defines the visual language of the Beauty Atelier website.

Every component, page, section, animation, spacing decision, and typography choice must follow this document.

If there is any conflict between generated code and this document, THIS DOCUMENT ALWAYS WINS.

---

# Project Goal

Create a premium luxury salon website.

The website should feel:

- Luxury
- Elegant
- Calm
- Minimal
- Editorial
- Spacious
- Modern
- Trustworthy

The design should never feel:

- AI Generated
- Generic
- Template-based
- Startup Landing Page
- SaaS Dashboard
- Corporate Website
- Bootstrap Theme

# Design Philosophy

Less is more.

Whitespace is part of the design.

Photography is more important than decoration.

Typography is more important than animations.

Luxury comes from restraint.

---

# Layout

Large spacing.

Never cram content.

Desktop Container

max-w-7xl

Readable text

max-w-3xl

Every section should breathe.

---

# Typography

Heading

Cormorant Garamond

Body

Inter

Rules

- Large headings
- Comfortable line height
- No tiny text
- Maximum two font families

---

# Color Palette

Background

#F8F6F2

Card

#FFFFFF

Primary Text

#111111

Muted Text

#777777

Gold Accent

Border

---

# Border Radius

Buttons

rounded-full

Cards

rounded-2xl

Images

rounded-3xl

---

# Shadows

Soft only.

No dramatic shadows.

---

# Buttons

Primary

Black background
White text

Secondary

White
Black Border

No gradients.

No glowing buttons.

---

# Cards

Large padding.

Minimal borders.

No glassmorphism.

No neon.

---

# Images

Photography is the hero.

Large images.

Natural lighting.

Eventually replace all placeholders with real salon images.

Never use unrealistic AI beauty models.

---

# Icons

Lucide Icons only.

---

# Animations

Allowed

- Fade
- Fade Up
- Text Reveal
- Image Zoom
- Hover Lift
- Smooth Scroll

Avoid

- Bounce
- Rotate
- Spin
- Crazy Parallax
- Heavy Motion

Animations should support the content.

Never distract.

---

# Components

Every component should

- be reusable
- be responsive
- have hover states
- use consistent spacing
- use semantic HTML

---

# Mobile

Mobile first.

Desktop should feel luxurious.

No hidden important content.

---

# Accessibility

Good contrast.

Keyboard friendly.

Semantic HTML.

Alt text for every image.

---

# Performance

Use next/image.

Lazy load images.

Avoid unnecessary JavaScript.

Optimize fonts.

---

# Code Style

Reusable components.

No duplicated code.

Use Tailwind utilities.

Avoid inline styles.

---

# Final Feeling

When someone opens the website they should think:

"This feels like a luxury boutique salon."

Not

"This looks AI generated."
